This is a classic command-line game that requires **Python 3** to be installed on your system.

### Running the Game

1.  **Clone the repository** to your local machine:
    ```bash
    git clone [https://github.com/nicola-empower/python-hangman-game.git](https://github.com/nicola-empower/python-hangman-game.git)
    ```

2.  **Navigate** into the project directory:
    ```bash
    cd python-hangman-game
    ```

3.  **Execute the script:**
    ```bash
    python hangman.py
    ```

4.  Follow the prompts to guess one letter at a time! You have a limited number of attempts to guess the hidden word.
